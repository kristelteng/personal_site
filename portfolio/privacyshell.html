Privacy Shell Inc.

  • April 2015
  • Web Dev | UX Design | Branding

Privacy Shell is a site where users learn the latest news about online privacy. It is also a service where users can connect with our privacy experts and browse through our suggested solutions and install the products that best suit their needs to have better control of their online privacy. My responsibilities were to

  • Create the UI/UX flow of the sites and the visual elements including the logo
  • Conducted user surveys, user research and user testing to understand the impact of the product on the users
  • Implemented analytics tracking and A/B testing to increase conversions
  • - Developed prototypes for multiple products using Invision

Tools used

  • Javascript
  • HTML
  • CSS
  • Adobe Suite
  • Wordpress
  • Google Analytics
  • Optimizely
