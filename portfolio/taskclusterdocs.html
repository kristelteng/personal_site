Taskcluster Documentation

  • May 2016
  • Web Dev

Taskcluster is a task execution framework that runs automated builds and tests. It consists of many small services.

  • Implemented an automated doc generation system, where each Taskcluster component updates triggers a new version of the static site to be generated.
  • It allows all developers on Taskcluster to have an automatic way to publish documentation when deploying new features.

Tools used

  • Node.js
  • ES6 Javascript
  • Babel
  • Markdown
  • Gulp
  • S3
