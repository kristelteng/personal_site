Privacy Snapper - User Dashboard

  • April 2015
  • Web Dev | UX Design

A user dashboard for Privacy Snapper. A browser extension that collects Privacy Policies of site that you visit.
Implemented a web scraping backend for legal documents using Scrapy. Designed the UI/UX flow for the user dashboard, profile page, subscription plans and user registration. Conducted user testing.
